# Personal Portfolio Website

This is a simple, responsive personal portfolio website built using HTML and CSS. It includes a homepage displaying project thumbnails, each with a hover effect that reveals a description. Clicking on a project takes the user to a dedicated page for that project.

## 🌐 Live Site
Your site will be live at:
```
https://yourusername.github.io
```
(Replace `yourusername` with your GitHub username)

## 🗂 Project Structure
```
├── index.html            # Main portfolio page
├── project1.html         # Individual project page 1
├── project2.html         # Individual project page 2
├── project3.html         # Individual project page 3
├── project1.png          # Project image 1
├── project2.png          # Project image 2
├── project3.png          # Project image 3
```

## 🚀 How to Use
1. **Clone or download** this repository.
2. **Replace** the placeholder images (`project1.png`, etc.) with your actual project screenshots.
3. **Edit** the HTML files to include your real project names, descriptions, and links.
4. **Push** to your GitHub repository named `yourusername.github.io`.
5. **Go to GitHub Pages** (Settings → Pages) and make sure GitHub Pages is enabled.

## ✨ Features
- Hover-to-reveal project descriptions
- Click-to-navigate individual project pages
- Clean, responsive layout
- Fully static site (no backend needed)

## 🛠 Tech Used
- HTML5
- CSS3
- GitHub Pages for deployment

## 📬 Customize It
- Add more projects by duplicating the `.project` block in `index.html` and adding new HTML pages
- Style it further using your own CSS or frameworks like Tailwind, Bootstrap, etc.
- Add a contact form, animations, or an "About Me" section
